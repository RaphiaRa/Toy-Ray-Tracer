### Toy-Tracer
### A simple software ray tracer written in C++.
Based on:
[_Ray Tracing in One Weekend_](https://raytracing.github.io/books/RayTracingInOneWeekend.html)

### Work in progress!

**Build Examples Instructions**
Dependencies:
- SDL2

```sh
$ mkdir build
$ cd build
$ cmake .. -DCMAKE_BUILD_TYPE=Release
$ make
```

**Run**  
```sh
./Example01
```
