### Toy-Tracer
### A simple software ray tracer written in C++.

### Work in progress!

**Build Examples Instructions**
Dependencies:
- SDL2

```sh
$ mkdir build
$ cd build
$ cmake .. -DCMAKE_BUILD_TYPE=Release
$ make
```

**Run**
```sh
./Example01
```
